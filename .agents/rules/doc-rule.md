# Quy tắc về vị trí đặt các file tài liệu docs, plans

NẾU KHÔNG CÓ YÊU CẦU RÕ RÀNG TỪ NGƯỜI DÙNG VỀ VỊ TRÍ ĐẶT FILE. CÁC TỆP LIÊN QUAN ĐẾN TÀI LIỆU VÀ PLAN TRIỂN KHAI FEATURE LUÔN ĐẶT TRONG FOLDER .agent/docs hoặc .agent/plans

## 1. Các tệp liên quan đến tài liệu (onboarding.md, tài liệu hướng dẫn, tài liệu kỹ thuật...) luôn được đặt trong folder .agent/docs/<tên-tài-liệu>

## 2. Các tệp liên quan đến plans (Brainstorm, kế hoạch triển khai tính năng, phân tích thiết kế kiến trúc,...) luôn được đặt trong folder .agent/plans/<tên-plan>

## 3. Nếu có yêu cầu rõ ràng từ người dùng về vị trí yêu cầu đặt file, hoặc file là file README.md hay các file khác thuộc về tài liệu tổng quan của dự án thì thực hiện theo yêu cầu của người dùng (bỏ qua rule ## 1 và rule ## 2)
